How to Use This Application

Messages

Click on the 'Messages' page to view a list of every message with a summary of its data. Through this page, you can also click on a message ID to view more information about that message.

Codes

Click on the 'Codes' page to view a list of every code.

Assign

Click on the 'Assign' page to be able to assign codes to parts of a specific message. To do this, you will need to enter the message's ID which can be found using the 'Messages' page.
